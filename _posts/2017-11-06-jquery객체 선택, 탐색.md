#### filter()

기본적으로 지원하지 않는 필터로 문서 객체를 선택할 때 사용한다.

~~~ html

<body>
  <h3>Header-0</h3>
  <h3>Header-1</h3>
  <h3>Header-2</h3>
  <h3>Header-3</h3>
  <h3>Header-4</h3>
  <h3>Header-5</h3>
</body>

<script>
  $(document).ready(function(){

        //사용 방법 1.
        $('h3:even').css({
          backgroundColor:'black',
          color:'white'
       });

        //사용 방법 2.
        $('h3').filter(':even').css({
            backgroundColor:'black',
            color:'white'
        });

        //사용 방법 3.
        $('h3').filter(function(index){
            return index%3 == 0;
        }).css({
            backgroundColor:'black',
            color:'white'
        });
  });
</script>

~~~


#### end()

문서 객체 선택을 한 단계 위로 올리는 메서드이다.

~~~ html

<body>
  <h1>Header-0</h1>
  <h1>Header-1</h1>
  <h1>Header-2</h1>
</body>

<script>
$(document).ready(function(){
  $('h1').css('background', 'orange').filter(':even').css('color', 'white').end().filter(':odd').css('color', 'red');
});
</script>

~~~

#### eq()

~~~ html

<body>
  <h1>Header-0</h1>
  <h1>Header-1</h1>
  <h1>Header-2</h1>
</body>

<script>
$(document).ready(function(){
  $('h1').eq(0).css('background', 'orange');
  $('h1').eq(-1).css('background', 'red'); //음수는 뒤에서 센다. -1은 뒤에서 첫 번째.
});
</script>

~~~

#### is()

~~~java

<body>
  <h1 class = "select">Header-0</h1>
  <h1>Header-1</h1>
  <h1 class = "select">Header-2</h1>
</body>

<script>
$(document).ready(function(){
  $('h1').each(function () {
    if($(this).is('.select')){
        $(this).css('background', 'orange');
    }
  })
});
</script>

~~~

#### parent()

$(this).parent()는 이벤트를 발생시킨 대상의 부모.

~~~ html

<body>
    <script>
      $(document).ready(function(){
          $('button').click(function(){
             $(this).text('비활성화하기');
             $(this).parent().css('background', 'red');
             $(this).parent().find('span').text('활성화');
          });
      });
    </script>

    <div>
      <span>비활성화</span>
      <button>활성화하기</button>
    </div>
    <div>
      <span>비활성화</span>
      <button>활성화하기</button>
    </div>
</body>


~~~
